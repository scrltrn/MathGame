Solve the Question is an educational math game that primarly focuses on teaching people of all ages about simple math calculations and math concepts
in a very easy way. Game contains several modes such as counting the digits of the pi, racing against a counter and just normal calculations such as adding,extracting,dividing and multiplying certain numbers
to match the ending result. It also has difficulty modes(easy, medium and hard) for player to decide which one they want to play.

Educational Concept of the Game:
The primary pedagogical goal of this game is to reinforce arithmetic fluency and mathematical problem-solving, specifically involving negative numbers and the order of operations.
Game applies the learning theory of constructivism, where players actively build their own understanding through experimentation.
The core mechanics directly facilitate this learning process; for example, the "Target Number" mode gives players a final answer and requires them to experiment with various numbers and operators to build a valid equation.
This reverse-engineering mechanic, paired with immediate visual feedback, teaches cognitive flexibility and allows players to learn through active trial and error.
